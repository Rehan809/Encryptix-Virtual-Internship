# Encryptix Virtual Internship

This repository contains Python projects completed as part of the Encryptix Virtual Internship program.
The projects focus on building logic, improving problem-solving skills, and understanding core Python concepts.

## Projects Included

### Calculator
A simple Python-based calculator that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

### Passcode Generator
A Python program that generates secure passcodes using random characters, helping understand randomness and security basics.

### To-Do List
A command-line to-do list application that allows users to add, view, and manage daily tasks efficiently.

## Technologies Used
- Python 3

## Objective
The goal of this internship was to gain hands-on experience with Python programming and develop small real-world applications.
